# EmojionEngine
EmojionEngine is a fun and interactive emoji library for JavaScript, designed to bring joy and creativity to your projects. 
With a simple and easy-to-use API, you can add engaging and expressive emojis to your web applications in no time!

Features 🎯
- 🎨 Fun collection of emojis, including the beloved 'pointer' emoji 👉

- 🏗️ Easy-to-use API for quick integration into your projects

- 📦 Lightweight with minimal dependencies






Usage :point_up_2:

To start using EmojionEngine, simply import it into your JavaScript file and start adding fun emojis to your projects!
```javascript
import EmojionEngine from './js/modules/emojionengine/src/core/emojionengine.js';

emojionEngine = new EmojionEngine({
    cursorSize : "30px",
    color : "#BADA55",
});
  
 ```
